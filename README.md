# 📡 Апаратно-програмний симулятор радіолокаційної обстановки

Цей проєкт — апаратно-програмний комплекс для цифрової обробки радіолокаційних сигналів (Radar Baseband Processor). Він імітує роботу приймального тракту РЛС, дозволяючи генерувати ЛЧМ-сигнали, імітувати відбиття від цілі та обробляти отримані дані за допомогою FPGA та мікроконтролера.

## 🛠 Апаратне забезпечення (Hardware)
* **FPGA:** QMTECH Intel Altera Cyclone 10CL025 (виконує всю високошвидкісну математику та генерацію сигналів).
* **MCU:** STM32F405RGT6 WeAct (виступає в ролі керуючого процесора).
* **Інтерфейс зв'язку:** SPI (STM32 виступає як Master, FPGA — як Slave).

## 🧩 Структура проєкту та Документація
Архітектура поділена на окремі апаратні модулі. Детальний опис кожного модуля, його портів та логіки роботи знаходиться в папці `docs/`:

*  **[Top-Level Модуль (radar_top.v)](docs/radar_top.md)** — Головний модуль, що об'єднує всі компоненти.
*  **[Генератор ЛЧМ-сигналу (dds_chirp.v)](docs/dds_chirp.md)** — Модуль Direct Digital Synthesis для генерації "пилки".
*  **[Імітатор віртуальної цілі (tx_generator.v)](docs/tx_generator.md)** — Модуль який об'єднує генерацію пилки і генерацію сінуса.
*  **[SPI Slave (SPI.v)](docs/spi_slave.md)** — Інтерфейс для прийому команд управління від STM32.
*  **[SPI Slave (sine_lut.v)](docs/sine_lut.md)** — Генерація сінуса.
