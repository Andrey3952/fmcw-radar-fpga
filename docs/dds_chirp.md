# Генерація фази для ЛЧМ-сигналу

* `phase` - Лінійний цифровий код поточного кута (від 0 до 65535).
* `FREQ_STEP` - Крок збільшення частоти сигналу.
* `freq_acc` - Акумулятор частоти, обмежена 1.25МГц щоб не було аліасингу.
* Тривалість ЛЧМ сигналу 32.76мкс.
* `phase_acc` - Акумулятор фази кут на одиничному колі.
